10. Assignment: Prime Number
take a number as input and determine if it is a prime number or not. 
Output "Prime" or "Not Prime" accordingly.

Algorithm:
-----------

Step 1 : Start
Step 2 : Declare the variables n,i,count.
Step 3 : initialize the i with 1 and count with 0 
Step 4 : Read the value of n
Step 5 : check i<=n then check  n%i==0 if yes increment count and increment i and otherwise stop
Step 6 : check  count==2 then print prime number otherwise print not prime number
Step 7 : Stop
